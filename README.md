# Frog Knight

This is the source code and general information for *Frog Knight*, a 3D puzzle sokoban adventure game built using the [Odin language](https://odin-lang.org/) and [Raylib](https://www.raylib.com/). Push blocks, solve puzzles, and explore charming levels as a heroic frog. This project started as a fun, learning-focused experiment in 3D development and is released here as a small, free version. In the future, there may be an expanded, fully-featured commercial release.

### Installation

- Make sure you have [Odin](https://odin-lang.org/docs/install/) installed.
- Clone this repo:
  ```
  git clone https://github.com/gbrferraz/frog-knight.git
  ```
- Build and run the project:
  ```
  odin run src
  ```

### Libraries Used

- [Raylib](https://www.raylib.com/): Simple and easy-to-use library to enjoy videogames programming.
- [Odin Language](https://odin-lang.org/): The entire codebase is written in Odin.

### Tools Used

- [Blender](https://www.blender.org/): For creating 3D models.

### Resources Used

- [Atkinson Hyperlegible](https://brailleinstitute.org/freefont): Main font used in-game.
- [Resurrect 64](https://lospec.com/palette-list/resurrect-64): Main color palette.
